# Python GUI Calculator

A fully functional calculator built with Python and Tkinter. This project demonstrates GUI programming, event handling, and internal state management in Python.

---

## Features

- **Graphical User Interface (GUI)** with responsive buttons and keyboard support.
- **Basic arithmetic operations**: addition, subtraction, multiplication, division.
- **Robust input handling**:
  - Supports decimal numbers
  - Backspace functionality
  - Clear (`C`) and Escape (`Esc`) to exit
  - Prevents invalid consecutive operators
- **Repeat last operation**:
  - Pressing `=` or Enter after a calculation repeats the last operation automatically (like real calculators).
- **Intelligent display formatting**:
  - Converts floats like `20.0` to `20` for a clean display.
- **Modular and object-oriented design**:
  - Clear separation of GUI, logic, and utility functions.
